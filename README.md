# PROGRAM
PROGRAM FOR PRINTING NATURAL NUMBER SERIES
CODE:
```
num = int(input())
for n_num in range(1,num+1):
    print(n_num)
```
INPUT:
9
OUTPUT:
1
2
3
4
5
6
7
8
9
